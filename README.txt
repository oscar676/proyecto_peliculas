Proyecto Realizado por Oscar Fonticoba Paleo

Descripción:
Aplicación web maverizada hecha en JAVA que realiza consultas de películas sobre una base de datos MYSQL.

La clase DbConnection contiene la configuración para la conexión a la base de datos.

Hay un script de la base de datos en la carpeta script, para crear la base de datos con unos datos de prueba.

Para desplegar el proyecto en el servidor generaremos un archivo .war y lo cargaremos en el tomcat.
